# 🧮 Calculator App

A simple and modern Android Calculator application built using **Kotlin**, **XML**, and **Material Design 3**. The app performs basic arithmetic operations with a clean and user-friendly interface.

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📱 Material Design 3 UI
* 🎨 Modern and Responsive Layout
* ⚡ Fast and Lightweight

## 🛠️ Built With

* Kotlin
* Android SDK
* XML Layouts
* Material Design 3 Components


## 🎯 Future Improvements

* Calculation History
* Dark Mode Support
* Scientific Calculator Functions
* Copy Result Feature
* Improved Input Validation
